Politica de Privacidad

Ultima actualizacion: 12 de mayo de 2026. Esta politica aplica a las apps TENKAI TENSHI que enlacen a esta pagina.

Resumen Datos locales Host & Client Archivos Apoya al creador Cambios futuros Contacto

Resumen

De forma general, las aplicaciones actuales de TENKAI TENSHI estan pensadas para funcionar de manera local u offline. No se utilizan servidores propios para vender, compartir o transferir datos personales ingresados por el usuario.

Aviso importante: si una app agrega cuentas, nube, anuncios, analiticas, pagos externos, SDKs o servidores, esta politica debera actualizarse.

Datos almacenados localmente

Las apps pueden guardar en el dispositivo informacion que el usuario introduce para operar la app: clientes, telefono, datos del negocio, materiales, impresoras, costos, impuestos, margenes, cotizaciones, facturas, abonos, ventas, devoluciones, rutas, usuarios, permisos y preferencias.

Uso de red local, Host & Client e intranet offline

Versiones profesionales pueden permitir que un dispositivo host se comunique con dispositivos cliente dentro de la misma red local. La finalidad es sincronizar facturas, registros, ventas, usuarios, permisos y datos operativos sin depender de internet.

Archivos, imagenes y documentos

Algunas apps pueden seleccionar logos, imagenes, archivos STL, documentos o generar PDFs. Estos recursos se usan para cumplir la funcion solicitada dentro de la app, salvo que el usuario decida exportarlos o compartirlos.

Apoya al creador y enlaces externos

Algunas apps pueden incluir enlaces externos a plataformas como Mercado Libre u otros servicios. Esos sitios operan bajo sus propias politicas. Algunos enlaces podrian ser de afiliado y generar una comision para el creador sin aumentar necesariamente el precio final para el usuario.

Cambios futuros

Si en el futuro se agregan servidores, cuentas, nube, publicidad, analiticas, SDKs, suscripciones externas o integraciones de terceros, esta pagina debera actualizarse para explicar esos cambios con claridad.
