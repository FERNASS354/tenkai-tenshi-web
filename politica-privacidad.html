Resumen

De forma general, las aplicaciones actuales de TENKAI TENSHI están pensadas para funcionar de manera local u offline. No se utilizan servidores propios para vender, compartir o transferir datos personales ingresados por el usuario.

Aviso importante: si una app agrega cuentas, nube, anuncios, analíticas, pagos externos, SDKs o servidores, esta política deberá actualizarse.
💾

Datos almacenados localmente

Dependiendo de la aplicación utilizada, el sistema puede almacenar configuraciones, clientes, productos, materiales, impresoras, costos, impuestos, márgenes, cotizaciones, facturas, abonos, ventas, devoluciones, rutas, usuarios, permisos y preferencias necesarias para el funcionamiento de la app.

📦

Respaldos y exportaciones manuales

Algunas aplicaciones permiten exportar configuraciones, bases de datos locales, documentos o respaldos manuales para proteger la información del usuario en caso de cambio de dispositivo, reinstalación o fallos del sistema. Los archivos exportados son administrados directamente por el usuario.

🌐

Uso de red local, Host & Client e intranet offline

Versiones profesionales pueden permitir comunicación entre un dispositivo host y dispositivos cliente dentro de la misma red local o intranet offline. Esta función busca sincronizar facturas, registros, ventas, usuarios, permisos y datos operativos sin depender de internet ni servicios en la nube.

📂

Archivos, imágenes y documentos

Algunas apps pueden seleccionar logos, imágenes, documentos, archivos STL, configuraciones técnicas o generar documentos PDF. Estos recursos se usan únicamente para cumplir funciones solicitadas dentro de la app, salvo que el usuario decida exportarlos o compartirlos manualmente.

🔗

Apoya al creador y enlaces externos

Algunas apps pueden incluir enlaces externos hacia plataformas como Mercado Libre u otros servicios. Dichos sitios operan bajo sus propias políticas de privacidad y condiciones de uso. Algunos enlaces podrían ser de afiliado y generar una comisión para el creador sin aumentar necesariamente el precio final para el usuario.

🧩

Variaciones entre aplicaciones

Cada aplicación del ecosistema TENKAI TENSHI puede implementar funciones distintas. Algunas versiones almacenan únicamente configuraciones básicas, mientras que otras pueden administrar operaciones empresariales, inventarios, usuarios, registros internos, producción, ventas, logística o distribución.

🚀

Cambios futuros

Si en futuras versiones se integran servicios en la nube, cuentas, sincronización online, SDKs externos, analíticas, pagos, anuncios o nuevas tecnologías, esta política de privacidad será actualizada para reflejar dichos cambios con transparencia.
