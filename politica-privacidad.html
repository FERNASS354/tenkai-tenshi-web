Legal y privacidad

Política de Privacidad

Aplicaciones desarrolladas por TENKAI TENSHI. Última actualización: 12 de mayo de 2026.

1. Resumen

Esta Política de Privacidad aplica a las aplicaciones desarrolladas por TENKAI TENSHI, incluyendo aplicaciones gratuitas y de paga enfocadas en cálculo de costos, cotizaciones, impresión 3D, administración de clientes, materiales, impresoras, facturas, abonos, archivos, distribución, redes locales y funciones relacionadas.

De forma general, las aplicaciones actuales de TENKAI TENSHI no utilizan servidores externos propios para recopilar, vender, compartir o transferir datos personales del usuario. La información ingresada por el usuario se utiliza para las funciones internas de cada aplicación y se guarda localmente en el dispositivo o dentro de una red local cuando la aplicación incluya funciones Host & Client.

Nota: Si en el futuro una app agrega cuentas, nube, analíticas, publicidad, pagos externos, base de datos remota o SDKs externos, esta política deberá actualizarse.

2. Aplicaciones cubiertas

Esta política puede aplicarse a TENKAI FORGE SUITE, TENKAI CALCULATOR 3D, TENKAI FORGE SUITE PRO, TENKAI DISTRIBUTION y otras aplicaciones publicadas por TENKAI TENSHI que indiquen esta política como su política oficial.

3. Información almacenada localmente

  • Nombre, teléfono u otros datos de clientes agregados por el usuario.
  • Datos del negocio configurados por el usuario.
  • Materiales, filamentos, impresoras, costos, extras, energía y parámetros de impresión.
  • Cotizaciones, facturas, abonos, saldos pendientes y registros relacionados.
  • Usuarios, roles, ventas, repartos, devoluciones o movimientos cuando la app incluya funciones de distribución o red local.
  • Logo, configuración, impuestos, márgenes, preferencias y ajustes internos.

4. Uso de red local, Host & Client e intranet offline

Algunas aplicaciones, como versiones profesionales Host & Client, pueden permitir conexión entre dispositivos dentro de una misma red local, intranet o red proporcionada por el dueño del host. En esos casos, la información puede compartirse entre el dispositivo host y los dispositivos cliente para sincronizar facturas, registros, ventas, usuarios, permisos y datos operativos.

Estas funciones están pensadas para operar sin internet. La información se mantiene dentro del entorno local del usuario, salvo que el propio usuario decida exportarla o compartirla por otros medios.

5. Uso de archivos, imágenes y documentos

Algunas aplicaciones pueden permitir seleccionar imágenes, logos, archivos STL, documentos o generar PDFs. Los archivos seleccionados se procesan localmente para cumplir la función solicitada.

6. Uso de la información

La información ingresada se utiliza para calcular costos, generar cotizaciones, crear PDFs, administrar clientes, controlar abonos, configurar impuestos, sincronizar datos locales, controlar repartidores, analizar devoluciones o mejorar el flujo de trabajo dentro de la aplicación.

7. Sección “Apoya al creador” y enlaces externos

Algunas aplicaciones pueden incluir una sección llamada “Apoya al creador”, “Filamentos recomendados”, “Materiales recomendados” o nombres similares. Esta sección puede mostrar enlaces externos a productos, listas o recomendaciones disponibles en plataformas de terceros, como Mercado Libre México u otros servicios externos.

Al tocar estos enlaces, el usuario puede salir de la aplicación y ser redirigido a una plataforma externa. Estas plataformas no son propiedad de TENKAI TENSHI y operan bajo sus propios términos, condiciones y políticas de privacidad.

Algunos enlaces pueden ser enlaces de afiliado. Esto significa que, si el usuario compra un producto después de entrar mediante uno de esos enlaces, TENKAI TENSHI o el creador de la aplicación podría recibir una comisión, sin que esto necesariamente aumente el precio final para el usuario.

8. Compartición de información

TENKAI TENSHI no vende, comparte ni transfiere datos personales ingresados en la aplicación a terceros mediante servidores propios. La información local permanece en el dispositivo del usuario o en su red local, salvo cuando el propio usuario decide compartir archivos, PDFs, cotizaciones o información mediante las opciones disponibles en su dispositivo.

9. Retención y eliminación

Los datos permanecen guardados mientras el usuario mantenga instalada la aplicación o hasta que los elimine manualmente, borre los datos de la app o desinstale la app.

10. Permisos

Se pueden solicitar permisos de archivos, galería, almacenamiento, documentos o acceso a internet para abrir enlaces externos. Las apps Host & Client pueden requerir permisos de red local para detectar o comunicarse con dispositivos dentro de la misma red.
